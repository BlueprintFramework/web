#### Version update notifications

Blueprint now tells you when extensions you have installed on your panel are outdated, useful right? It uses the [blueprint.zip/browse](/browse) API for version data.

#### Node.js version update

We're switching everyone over to Node.js v22 or later. If you are on an older version, please switch to a newer one :)
